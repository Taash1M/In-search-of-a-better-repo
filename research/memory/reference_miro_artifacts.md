## Convention
- One subfolder per Miro board/project
- Within each: `architecture/` for Azure-icon PNGs, `dataflow-d2/` for D2 sources, `dataflow-svg/` for rendered SVGs
- Miro MCP has no image upload — drag-drop PNGs/SVGs onto boards manually
