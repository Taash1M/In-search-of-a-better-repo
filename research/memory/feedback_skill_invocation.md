---
name: Auto-invoke skills
description: Always activate the right skill(s) automatically without asking — FYI the user which skills are active as work progresses. User may redirect or add skills.
type: feedback
---

Always invoke the necessary skill(s) proactively based on the task, then inform the user which skills are active as work progresses. Do NOT ask "should I use X skill?" — just activate it.

**Why:** User prefers autonomous skill selection to keep momentum. They will redirect or suggest additional skills if needed, but the default should be action, not permission-seeking.

**How to apply:** When a task matches one or more skill triggers, invoke them immediately. Mention which skills are active in a brief FYI as you go (e.g., "Using /paperclip and /excel-create for this"). If multiple skills apply, activate all of them.
